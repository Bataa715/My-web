# **App Name**: LinguaCore

## Core Features:

- Home Page Display: Display web name, logo, introduction, navigation menu, call-to-action button and dark/light toggle on the home page.
- About Me Page: Display personal information, including avatar, name, profession/goals, reasons for creating the site, and learning goals.
- English Vocabulary Table: Create, edit, and delete English vocabulary words with Mongolian meanings, stored in local storage with a memorized checkbox.
- English Grammar Cards: Display grammar rules with explanations and example sentences, using an accordion/toggle UI component.
- Japanese Character Grids: Display Hiragana and Katakana grids with visual highlights for memorized state, stored in local storage.
- Japanese Vocabulary Table: Create, edit, and delete Japanese vocabulary words with Romaji and Mongolian meanings, stored in local storage with a memorized checkbox.
- Programming Concepts Display: Display programming concepts like JavaScript, HTML/CSS, and React, with short explanations per topic.
- Cheat Sheet with Code Snippets: Display code snippets and syntax blocks, including a copy button for easy copying of the code.
- Programming Progress Tracker: Track learning progress for programming concepts using checkboxes, stored in local storage.

## Style Guidelines:

- Primary color: Soft blue (#A0D2EB) to evoke a sense of calmness and learning.
- Background color: Light gray (#F0F4F8), offering a clean and modern look that emphasizes content.
- Accent color: Pale yellow (#E8DAB2) to highlight key interactive elements while staying accessible and clean.
- Body and headline font: 'Inter' sans-serif font known for its legibility and modern appearance, suitable for both headlines and body text.
- Code font: 'Source Code Pro' monospace font, ideal for displaying code snippets.
- Use simple emoji icons to visually represent each feature for intuitive navigation.
- Design a responsive layout, starting mobile-first, transitioning to a desktop grid.
- Apply smooth hover effects on interactive elements, offering delightful and clean micro-interactions.
